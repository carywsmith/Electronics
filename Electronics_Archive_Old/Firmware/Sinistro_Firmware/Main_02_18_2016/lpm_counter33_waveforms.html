Sample behavioral waveforms for design file lpm_counter33.tdf

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design lpm_counter33.tdf. The design lpm_counter33.tdf is a 8 bit down counter. The counter supports a synchronous load of data input. The counter has an asynchronous set signal, that sets the counter to all 1's.
