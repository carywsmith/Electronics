Sample behavioral waveforms for design file lpm_counter32.v

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design lpm_counter32.v. The design lpm_counter32.v is a 13 bit down counter. The counter supports a synchronous load of data input. The counter has an asynchronous set signal, that sets the counter to all 1's.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.
